# StopWordApi

## But de cette API

**Cette Api a pour but de fournir un service gratuit pour filtrer un texte et en ressortir le même texte sans StopWords**

## Routes disponibles ##

```
POST /texts

body requis
{
    "data": "text à filtrer"
}
```

cette api est pour l'instant disponible via l'url
``https://stop-word-api.herokuapp.com``
